Using Menu Commands

Below here is the explanation of the various commands.

Actions:

  • New - creates a new grid in a new tab
  • Import - loads a grid from a file
  • Clear - erases the current grid
  • Clear Sparks - erases the current grid from all the sparks
  • Clear Links - erases the current grid from all the links
  • Export - saves the current grid in a file
  • Save - saves the current grid status
  • Quit - to exit from the application

Edit:

  • Undo - undoes the last action
  • Redo - redoes the previous action
  • Cut - cuts the entities from the current selection
  • Copy - copies the entities from the current selection
  • Paste - pastes some entities from the clipboard to the grid
  • Flip horizontally - saves the current grid in a file
  • Flip vertically - saves the current grid status
  • Rotate 90 deg - rotates by 90 degrees the current selection
  • Rotate 180 deg - rotates by 180 degrees the current selection
  • Rotate 270 deg - rotates by 270 degrees the current selection

Environment:

  • Smart selection - enables/disables smart selection
  • Minimap - enables/disables minimap
  • Chess creation - enables/disables chess method for the creation of entities
  • Play - starts the simulation on the current grid
  • Step - executes only one step of the simulation for che current grid
  • Stop - stops the simulation on the current grid

Items:

  • Spark - inserts entities of Spark type
  • Arrow UP - inserts entities of Arrow UP type
  • Arrow DOWN - inserts entities of Arrow DOWN type
  • Arrow RIGHT - inserts entities of Arrow RIGHT type
  • Arrow LEFT - inserts entities of Arrow LEFT type
  • 1D Automata - inserts entities of 1D Automata type (with rule 30 as default)
  • Link IN - creates a link of type IN
  • Link OUT - creates a link of type OUT
  • Links Manager - opens the links manager dialog

Info:

  • Cellular Automata Manager - opens a dialog with info about this program
  • Help - opens the current guide
