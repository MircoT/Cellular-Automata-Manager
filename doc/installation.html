Installation

To install the package simply launch from the main directory the command:

  • python setup.py install

If you want to create a source distribution package use:

  • python setup.py sdist


The application needs the following dependencies to run:


  • wxPython - http://www.wxpython.org/download.php#stable
  • pygame - http://www.pygame.org/download.shtml
  • six - https://pypi.python.org/pypi/six

To run the application without install it, you can use this command from the main directory:

  • python main.py

After the installation, you can run the application simply using:

  • cam-env

Notes: if you are under windows, the Script folder must be on your PATH variable.
On Linux, if you can't start the application after a correct installation, try
to use the command which cam-env to search the executable.
